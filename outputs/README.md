# outputs/

This directory holds compiled PDFs and private application tracking data.

## What lives here (locally — gitignored)

- `*.pdf` — compiled resumes (regenerated by `../build_resumes.sh` or `..\build_resumes.ps1`)
- `job_application_tracker*.csv` — application tracker (domestic + international)
- `job_batch_*.md` — per-batch application notes
- `job_search_shortlist.md` — shortlist of target companies
- `job_application_action_plan.md` — overall action plan
- `cover_notes/` — per-company cover letter notes (`*.txt`)
- `coverletter.docx` — cover letter template
- `resume_variant_playbook.md` — variant selection guide

## For the open-source repo

Everything in this directory is gitignored except this README.

After cloning the public repo, run `../build_resumes.sh` (macOS/Linux) or `..\build_resumes.ps1` (Windows) to regenerate the PDFs locally. See `../docs/BUILD_MAC.md` for TeX install instructions.

## Iron rule

Never copy `.tex`, `resume.cls`, or `fontawesome` files into this directory. Only PDFs and tracking documents belong here.
